Program description:
Note: The libraries that the python program depends on are installed one by one
How to run the program: After installing python and all dependent libraries, you can run it in the following two ways. (1) Run python ***.py; (2) Install pycharm and configure it
After the environment, you can run.
(1) cluster_superalloy.py: 266 alloy creep samples were classified into cluster categories to distinguish the differences in creep mechanisms under different compositions, heat treatment processes and microstructures;
(2) ClusterResultVisualization.py: visualization of cluster division;
(3) DCSA_analysis.py and OptModelForCluster.py: adaptive selection of the optimal prediction model;
(4) DCSA_ModelValidation.py: The predictions of the 8 newly collected alloy sample models.

程序说明：
注：python程序依赖的库一一装好
运行程序方法：安装好python及所有依赖库后，可通过以下两个方式运行。(1) python ***.py运行;（2）安装pycharm，配置好
环境后，即可运行。
（1）cluster_superalloy.py :对266条合金蠕变样本进行簇类别划分，以区别不同成分、热处理工艺和微观结构下
的蠕变机制的差异性；
（2）ClusterResultVisualization.py：聚类划分可视化；
（3）DCSA_analysis.py和OptModelForCluster.py：最优预测模型的自适应选择；
（4）DCSA_ModelValidation.py：新收集的8条合金样本模型的预测。
